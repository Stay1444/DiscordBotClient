# DiscordBotClient

A Discord Bot Client GUI written in WPF. Shitty code and slow as fuck. 

Features:
- [x] List Servers
- [x] List Text channels
- [x] Send and receive messages
- [x] Manage messages
	- Edit Messages
	- Delete Messages
- [x] Manage channels
- [x] View members with their role color
- [x] Infinite scrolling
